# Samir Metal & Tubes - Official Website

A modern, responsive website for Samir Metal & Tubes, a leading importer and stockist of seamless pipes and tubes established in 1991.

## About the Company

Samir Metal & Tubes has been providing quality non-ferrous and ferrous materials to major industries across India since 1991. We specialize in:

- Carbon Steel Seamless Pipes & Tubes
- Alloy Steel Seamless Pipes & Tubes
- Stainless Steel Seamless Pipes & Tubes
- Low Temperature Seamless Pipes & Tubes
- Heat Exchanger, Condenser & Boiler Tubes

## Website Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design suitable for B2B industry
- **Contact Form**: Integrated enquiry form with email functionality
- **Product Showcase**: Detailed specifications and grades for all products
- **Quality Certifications**: Information about certifications and inspection agencies
- **SEO Optimized**: Meta tags and structured content for better search visibility

## Technical Stack

- HTML5
- CSS3 (with modern features like Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (Inter)

## Local Development

To run this website locally:

1. Clone the repository
2. Open `index.html` in your web browser
3. For live reloading during development, use any local server like Live Server extension in VS Code
